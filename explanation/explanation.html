PDFファイルの暗号化・復号ツール

このツールでは、PDFファイルの暗号化および復号を行うことができます。

使用方法:

  1. PDFファイルの選択: 「暗号化」または「復号」ボタンをクリックし、処理したいPDFファイルを選択します。
  2. パスワードの入力: USER パスワードを入力し、確認のために再度パスワードを入力します。
  3. 処理の実行: 「暗号化」ボタンまたは「復号」ボタンをクリックして、処理を開始します。

入力ファイル形式: 任意のPDFファイル (.pdf)

出力形式:

  • 暗号化時: ファイル名は ANY_encrypted.pdf (例: foo.pdf → foo_encrypted.pdf)
  • 復号時: ファイル名は ANY_decrypted.pdf (例: foo.pdf → foo_decrypted.pdf)

JPGファイル名変更ツール

このツールでは、撮影日により, JPGファイルの名前を変更することができます。

使用方法:

  1. フォルダーの選択: 「参照」ボタンをクリックし、JPGファイルが含まれるフォルダーを選択します。
  2. 処理の開始: 「処理開始」ボタンをクリックして、ファイル名変更処理を開始します。

入力ファイル形式: 任意のJPGファイル (.jpg)

出力形式: ファイル名は [通し番号] MMDD-hhmmss.jpg の形式になります。

通し番号は 0, 1, 2, ... / 撮影日の昇順です (過去のほうが 番号が若く, 新しい方が 番号が大きい)

例: foo.jpg → [0] 0101-231034.jpg
